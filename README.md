# My-dotfiles
### All my config files
## Usage
First , install vim-plug : https://github.com/junegunn/vim-plug
and : 
```
git clone https://github.com/mo1ein/My-dotfiles.git
cp My-dotfiles/.vimrc ~/ && cp My-dotfiles/.tmux.conf ~/
```
 Then go to vim and type : ``` :PlugInstall```
